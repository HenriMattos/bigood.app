# Assets do Bigood

## Estrutura

- `public/brand`: logos, marcas e icones usados pela aplicacao.
- `public/brand/source`: arquivos originais enviados pelo designer.
- `public/images/landing`: imagens comerciais da landing page.
- `public/images/portal`: imagens das telas de login e portal.
- `public/images/legacy`: assets antigos mantidos apenas para compatibilidade.
